Preguntas

Dar un par de ejemplos de uso de etiquetas meta.

Las etiquetas meta por lo general llevan dos atributos, uno que indica el tipo de información y otro el contenido. Se puede utilizar para indicar el código de lenguaje al navegador, indicando por ejemplo al primer valor "Content-type" y al segundo el valor "UTF-8". También se puede utilizar para dirigir al usuario a otra pagina al refrescarla, colocando "refresh" en el primer atributo y en el segundo "5;url=http://www.youtube.com" (siendo 5 la cantidad de segundos que tarda en redirigir). Otro ejemplo es colocando "description" en el primer atributo y luego en el otro una descripcion de la página.

¿Como se relaciona un header http con la etiqueta head de HTML?

Se puede relacionar a través del atributo http-equiv que proporciona un header HTTP para la información/valor del contenido del atributo.

¿Que es UTF8?

UTF-8 (Unicode Transformation Format-8) es un formato de codificación de caracteres, en el que cada caracter es representado por un octeto (8 bits). Cada carácter es identificado por un número, pero para que ese número o código sea entendido por la máquina debe estar codificado, es decir traducido a binario de un modo específico. Unicode utiliza diferentes formas de codificación, entre ellas UTF-8.

¿Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?

El atributo style sirve para aplicar estilo, su valor es una cadena de parámetros de estilo nombre:valor;nombre:valor;….Cada parámetro indica una variable de estilo. Es aplicable a las etiquetas body, p, h1, h2, hr, img src, ul, li, section, articule, footer (por nombrar algunas vistas en el PDF).

¿Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?

Con target="_blank", este atributo indica que el nuevo documento será abierto en una nueva pestaña del navegador. El mismo se coloca junto con href dentro de la etiqueta "a"

Explicar cuando usarían etiquetas semánticas y con que propósito.

Cuando sea necesario (para organizar/estructurar el documento), con el fin de indicar el contenido que engloban y su relación con el conjunto de elementos del documento HTML.

¿Que atributos de estilo básicos aplicarían a una tabla? Explicar

El atributo border para poder mostrar bordes más sólidos, width y height para el alto y ancho, background-color para poner un color de fondo.

¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

El body en HTML es la etiqueta usada para indicar el cuerpo de un documento HTML, es la parte donde se describe el contenido de la página (su estructura, su forma, sus colores, texto, y todo lo visual) mientras que en el body del protocolo HTTP contiene datos, pueden ir datos ingresados por el usuario en un formulario y archivos que el cliente quiera enviar al servidor
